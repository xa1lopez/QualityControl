const std::string  checkClassName,
const std::string  checkLibraryName = ""

Add a check to be executed on this object when computing the quality. If a check with the same name already exists it will be replaced by this check. Several checks can be added for the same check class name, but with different names (and they will get different configuration).

Author
Barthelemy von Haller
Parameters
nameArbitrary name to identify this Check.
checkClassNameThe name of the class of the Check.
checkLibraryNameThe name of the library containing the Check. If not specified it is taken from already loaded libraries.

Quality o2::quality_control::core::MonitorObject::getQuality ( ) const

Get the quality of this object.

The method returns the lowest quality met amongst all the checks listed in mChecks. If there are no checks, the method returns Quality::Null.
